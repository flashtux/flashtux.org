== Install

=== Dependencies

Following packages are *required*:

* python >= 3.6
* python-django >= 1.11
* python-tz
* python-django-countries
* PostgreSQL.

=== Deploy

Run the install script and follow instructions:

----
$ ./bin/install.sh
----

Run Django server:

----
$ ./test.sh
----

And just point your browser on <http://127.0.0.1:8000/>, that's all!
